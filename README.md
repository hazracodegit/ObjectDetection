# AI Object Detection

An AI-powered web application that detects and identifies objects in images using computer vision and deep learning techniques. Users can upload images through a modern web interface and receive object detection results with confidence scores.

## Features

- Upload images for object detection
- Real-time object recognition
- Bounding box visualization
- Confidence score display
- Responsive and modern user interface
- Fast image processing
- Easy-to-use web application

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- Flask

### AI & Computer Vision
- OpenCV
- YOLO (You Only Look Once)
- NumPy
